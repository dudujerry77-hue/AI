# templates/

Reusable document templates so that every phase file, session log, ADR, PR, and review is structured consistently regardless of which agent or human produces it.

## Contents

- `phase-template.md`
- `session-log-template.md`
- `adr-template.md`
- `pr-template.md`
- `code-review-template.md`

Copy the relevant template, fill it in, and save it in the appropriate directory (`phases/`, `sessions/`, `decisions.md`, PR description, `reviews/`).
